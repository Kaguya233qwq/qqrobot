# qqrobot
## 基于go-cqhttp的qq机器人

## 环境安装

下载python3.8以上版本

安装依赖

~~~python
pip install flask
pip install requests
~~~



## 部署方法

### 1.首先打开qq_login 双击go-cqhttp.exe 然后打开go-cqhttp.bat文件。选择0 的http 协议，然后在生成的config.yum里面填写机器人QQ号 然后双击go-cqhttp.bat 运行用机器人qq扫码登录。

### 2.然后到项目根目录打开cmd 输入

~~~cmd
python.exe main.py
~~~
